# 4 欄網格布局指南

本文件說明 DIDI DADA 超級商城的 4 欄網格系統，以 **Full HD (1920x1080)** 為基準設計。

## 📐 網格系統概述

### 設計理念

網格系統採用 **響應式 4 欄布局**，根據螢幕寬度自動調整欄位數量：

| 解析度範圍 | 網格欄位 | 間距 | 適用裝置 |
|-----------|---------|------|---------|
| **1920px+** | **4 欄** | **2.5rem** | **Full HD 桌面** ⭐ |
| 1366px - 1919px | 3 欄 | 2rem | HD 桌面 |
| 1024px - 1365px | 3 欄 | 1.5rem | 筆記型電腦 |
| 768px - 1023px | 2 欄 | 1.5rem | 平板 |
| 481px - 767px | 2 欄 | 1.25rem | 大型手機 |
| 480px 以下 | 1 欄 | 1rem | 手機 |

## 🔧 CSS 實現

### 基礎網格樣式

```css
.items-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: 2rem;
  margin-bottom: 3rem;
}
```

### 響應式斷點

#### Full HD (1920px+) - 4 欄

```css
@media (min-width: 1920px) {
  .items-grid {
    grid-template-columns: repeat(4, 1fr);
    gap: 2.5rem;
  }
}
```

**視覺效果**：
- 每個卡片寬度：約 320px
- 間距：40px (2.5rem)
- 總寬度：1400px（容器最大寬度）
- 每行 4 個項目，視覺平衡

#### HD (1366px - 1919px) - 3 欄

```css
@media (min-width: 1366px) and (max-width: 1919px) {
  .items-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}
```

**視覺效果**：
- 每個卡片寬度：約 360px
- 間距：32px (2rem)
- 總寬度：1200px
- 每行 3 個項目

#### 筆電 (1024px - 1365px) - 3 欄

```css
@media (min-width: 1024px) and (max-width: 1365px) {
  .items-grid {
    grid-template-columns: repeat(3, 1fr);
    gap: 1.5rem;
  }
}
```

**視覺效果**：
- 每個卡片寬度：約 290px
- 間距：24px (1.5rem)
- 總寬度：960px
- 每行 3 個項目

#### 平板 (768px - 1023px) - 2 欄

```css
@media (min-width: 768px) and (max-width: 1023px) {
  .items-grid {
    grid-template-columns: repeat(2, 1fr);
    gap: 1.5rem;
  }
}
```

**視覺效果**：
- 每個卡片寬度：約 350px
- 間距：24px (1.5rem)
- 總寬度：100%（流式布局）
- 每行 2 個項目

#### 大型手機 (481px - 767px) - 2 欄

```css
@media (min-width: 481px) and (max-width: 767px) {
  .items-grid {
    grid-template-columns: repeat(2, 1fr);
    gap: 1.25rem;
  }
}
```

**視覺效果**：
- 每個卡片寬度：約 220px
- 間距：20px (1.25rem)
- 總寬度：100%
- 每行 2 個項目

#### 手機 (480px 以下) - 1 欄

```css
@media (max-width: 480px) {
  .items-grid {
    grid-template-columns: 1fr;
    gap: 1rem;
  }
}
```

**視覺效果**：
- 每個卡片寬度：100%
- 間距：16px (1rem)
- 總寬度：100%
- 每行 1 個項目（垂直堆疊）

## 🎨 設計優勢

### 1. 視覺平衡

**4 欄布局優勢**：
- 充分利用 Full HD 寬螢幕空間
- 每個卡片有足夠展示空間（320px）
- 間距適中（40px），不擁擠也不稀疏
- 2 行 × 4 欄 = 8 個項目，視覺完整

### 2. 內容密度

**適中的資訊密度**：
- 單屏可顯示 8 個項目（無需滾動）
- 每個卡片包含：圖片 + 標題 + 描述 + 連結
- 用戶可快速瀏覽所有推薦項目

### 3. 響應式適配

**平滑過渡**：
- 1920px → 4 欄（最佳）
- 1366px → 3 欄（標準）
- 768px → 2 欄（平板）
- 480px → 1 欄（手機）

**無斷層感**：
- 使用相對單位（rem）
- 間距隨解析度調整
- 卡片寬度自動計算

## 🔄 內容更新指南

### 添加新項目

1. **複製現有卡片結構**

```html
<!-- 項目 9 -->
<div class="item-card">
  <img src="https://via.placeholder.com/300x200?text=新商品" alt="新商品" class="item-card-image">
  <div class="item-card-content">
    <div>
      <span class="item-card-category">分類</span>
      <h3 class="item-card-title">商品名稱</h3>
      <p class="item-card-description">商品描述...</p>
    </div>
    <a href="#" target="_blank" class="item-card-link">查看詳情</a>
  </div>
</div>
```

2. **調整網格顯示**

**9-12 個項目**：
- Full HD：3 行 × 4 欄
- HD：4 行 × 3 欄
- 平板：5-6 行 × 2 欄

**13-16 個項目**：
- Full HD：4 行 × 4 欄
- HD：5-6 行 × 3 欄
- 平板：7-8 行 × 2 欄

### 減少項目

**4 個項目**：
- Full HD：1 行 × 4 欄（完美）
- HD：2 行（1 行 3 個 + 1 行 1 個）
- 平板：2 行 × 2 欄（完美）

**6 個項目**：
- Full HD：2 行（1 行 4 個 + 1 行 2 個）
- HD：2 行 × 3 欄（完美）
- 平板：3 行 × 2 欄（完美）

**建議**：保持項目數量為 **4 的倍數**（4、8、12、16），以在 Full HD 下完美顯示。

## 🧪 測試建議

### 1. 瀏覽器開發者工具

**Chrome DevTools**
1. 按 `F12` 開啟開發者工具
2. 按 `Ctrl+Shift+M` 切換裝置模式
3. 測試以下解析度：
   - **1920x1080** ⭐ 預設基準（4 欄）
   - 1366x768（3 欄）
   - 1024x768（3 欄）
   - 768x1024（2 欄）
   - 375x667（1 欄）

### 2. 響應式測試頁面

訪問 `responsive-test.html` 查看即時資訊：
- 目前解析度
- 當前斷點
- 網格欄位數
