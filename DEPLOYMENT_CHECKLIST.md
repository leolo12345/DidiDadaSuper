## GitHub Pages 部署步驟

### 倉庫設定
- [ ] 在 GitHub 建立 `DidiDadaSuper` 倉庫
- [ ] 倉庫設定為 Public
- [ ] 初始化 Git 並推送程式碼
- [ ] 確認所有檔案已推送到 `main` 分支

### Pages 設定
- [ ] 進入 Settings → Pages
- [ ] Source 設定為「Deploy from a branch」
- [ ] 分支選擇 `main`
- [ ] 目錄選擇 `/ (root)`
- [ ] 點擊 Save 並等待部署完成

### 部署驗證
- [ ] 訪問 `https://YOUR_USERNAME.github.io/DidiDadaSuper/`
- [ ] 首頁正常顯示
- [ ] 所有分類頁面可訪問
- [ ] 圖片正確顯示
- [ ] 樣式正確應用
- [ ] 連結有效

## 自訂域名配置（可選）

### GitHub 設定
- [ ] 進入 Settings → Pages
- [ ] 在「Custom domain」欄位輸入 `www.DidiDadaSuper.com`
- [ ] 點擊 Save
- [ ] 確認 `CNAME` 檔案已自動建立

### DNS 設定
- [ ] 登入域名提供商
- [ ] 設定 DNS 記錄（CNAME 或 A 記錄）
- [ ] 等待 DNS 生效（24-48 小時）
- [ ] 驗證 DNS 設定：
  ```bash
  nslookup www.DidiDadaSuper.com
  ```

### 域名驗證
